众所周知，90年代深度学习沉寂，很大的一个原因是，当深度增大时，会出现梯度消失问题，导致网络难以收敛，当时没有非常有效的解决手段。当然，今天我们也面临着同样的问题，但由于对梯度消失问题的深度理解，提出了很多有效的方案，其中LSTM尤其出名。

下面进行手推公式，公式只详细阐明梯度消息和保障逻辑，不会进行各种矩阵的角标变化等详细说明，主要分成三部分：

1. 详细推导前馈神经网络（全连接和卷积神经网络）中的梯度消失问题，这是真梯度消失问题，影响收敛；
2. 详细推导循环神经网络中梯度消失问题，其与前馈神经网络有很明显不同，是个假梯度消失问题，影响序列记忆长期历史；
3. 详细推导LSTM，说明其是如何解决RNN的梯度消失问题的。
