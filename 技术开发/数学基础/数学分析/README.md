## 

我眼中的数学

数学不是自然科学，类似于宗教可以自行演化一个世界。现代数学基于ZFC（策梅洛-弗兰克尔集合论）公理系统，通过定义概念、定义运算，采用演绎推理获得了一些有意义的引理、命题、定理等，这些定理浓缩着智慧，即可指导实践也可为深入研究铺路。

在计算机科学领域，有一句话叫没有什么是加一层解决不了的问题。在我看来，数学就是现实与思维之间的间接层，现实实例很多——一棵树、一个人、一本书，一棵树 “加”一个树是两个树……如果需要人脑对每种实例都建立一套描述体系，那么还不把人累死。所以抽象出了 自然数 1 用来描述所有实例的个数概念，抽象出了 + 来表示数的相加。自然数系的出现只是简单的抽象，而数学体系内部会有现有概念的更高层、更复杂的抽象。不过就像coding一样，并不是接口层越抽象越好，数学本身内部的更高层级抽象也难说好坏，更多思考可以看这个问题[什么是好的数学](https://www.zhihu.com/question/36993450/answer/138034109)。

回到工科必修的数学分析上来，在我眼里数学分析就研究了两个对象 1. 集合 2.映射，具体说来就是：

- 集合的结构和性质，结构如元素构成，性质如可数性、有界性、完备性等。
    
- 映射的结构与性质，结构如XX类型集合到XX类型集合，性质如连续性、可微性等。
    

这两者既相互独立又相互交织，其实线性代数也是如此，研究对象只有线性空间（集合）和线性变换（映射）。

## 

1. 集合的结构与性质

集合的抽象层次太高，而工科需要的数学分析要与实际结合，能解决实际工程问题，所以数学分析具体讲的是两个具体的集合对象—数集和欧几里得空间，其是工程问题的直接抽象，相信大家对这一点都深有感触。

- 数集（N、R等），从满足Peano公理的自然数集到通过有理数柯西序列定义的实数集，数集不断扩充，其性质也变得更为丰富。
    
- 欧几里得空间（Rn），形如(x1,...,xn)的有序元组构成的集合，多元函数微积分依赖的集合。
    

数集和欧几里得空间更高层的统一概念是度量空间，也即定义了距离函数（度量）的集合。对于集合X，度量定义为d:=X×X→[0,∞)，换句话说对于任意x,y∈X，有d(x,y)≥0且d要满足正性、对称性和三角不等式等性质。对于度量空间还有更高层的拓扑空间抽象，这里就不展开了，也用不到。

数集上的标准度量是绝对值 d(x,y)=|x−y|。

欧几里得空间的度量常用的有3种：

- l2度量，dl2((x1,...,xn),(y1,...,yn))=(x1−y1)2+...+(xn−yn)2
    
- l1度量，dl1((x1,...,xn),(y1,...,yn))=|x1−y1|+...+|xn−yn|
    

  

- l∞度量，dl∞((x1,...,xn),(y1,...,yn))=sup{|xi−yi|:1≤i≤n}
    

  

有了度量就可以定义更多的概念，下面只谈与映射不相关的概念，其是后续映射研究的基础。比较重要的概念包括：

  

1. 附着点，内点、外点、边界点等概念，通过度量定义的；
    
2. 开集和闭集，如数域上区间，闭集包括所有边界点，开集不包括任何边界点；
    
3. 有界性，集合是否有界；
    

## 

2. 映射的结构

映射、函数、变换、态射等等都是在不同集合上讨论时用的相同概念，这些概念抽象等级太高，数分里研究的集合就两种，对应的映射类型其实也就以下四种：

1. 序列（数列），自然数集 到 实数集的映射；
    
2. 一元函数， 实数集 到 实数集的映射；
    
3. 多元函数，欧几里得空间 到 实数集的映射；
    
4. 函数序列，自然数集 到 函数空间（定义为集合到集合的映射构成的集合）的映射。
    

序列或函数这些映射本身就是对象，也可以定义对象具有的普遍性质，重要的包括：

- 相等，相等概念需要满足自反、对称、传递和代入四个公理；
    
- 运算，重要的有4种： 加减乘除（分母不为零）、数乘 、max和min、复合。
    

  

## 

3. 映射的什么性质值得研究？

有一个数列，那么序列的什么方面值得研究呢？现实中常常有这么一种序列，随着不断增大，两个元素越来越接近，基于这样一种事实，引入了数分乃至整个分析体系的一个很重要的概念——极限。

还记得前面提到了度量概念吗？有了度量才能定义极限，因为极限语言（或）表述无限接近这种大白话时需要用度量来严格表述。比如序列的极限定义：对于任意，都存在一个，使得时，都接近于一个数，接近意味着。此时记作，就是所谓的序列极限。函数也可以采用类似的方式来定义函数极限，其采用的极限语言则是。

有些人觉得序列极限和函数极限不同，其实不是的，是一样的东西，函数极限可以转用序列极限语言来表述的。在更高层的集合抽象中，比如拓扑空间，这些形状各异的极限都归结于一种。

如果序列有极限，极限是唯一的。极限表面上是一个对象、一个值，而本质上则是用一个新的概念来表述某个序列无限接近的动态性质。

对于极限这个对象本身，也可以定义对象具有的一些普遍性质：

- 运算，重要的有3种： 加减乘除、数乘 、max和min。
    

至此，引出了数分中的基本概念—极限，后续的级数、微积分和Fourier级数等也都可以逐个亮相了。

  

## 

4. 定义了极限的四种映射

数学学不好，最大的症结在于对基本概念的认知不清，包括对其定义本身的认知不清以及此概念所属范围的认知不清，前者导致用不对后者导致不知道何时用。这一部分，也不会深入细节，主要目标是将数分中不同的概念分门别类，知道它从何而来，可属于哪一块。

## 

4.1 序列（数列 N→R）

有了极限概念之后，就可以研究一些特殊的序列以及集合的更多性质，主要包括以下概念：

- 柯西序列，一种在自身元素之间用ϵ−N语言定义的序列。
    
- 集合的完备性，如果集合的柯西序列都收敛，则是完备的，实数集是最小的完备数集。
    
- 集合的紧致性，如果集合的任何序列存在一子序列收敛，则是紧致的，实数集上的闭区间是紧致的。
    
- 级数，也即序列的和。将级数转化为研究 序列前n项和 构成的序列，而序列我们是熟悉的。。
    

## 

4.2 一元函数（R→R）

在极限基础上，函数上可以定义一些新概念。事实上，数分里就讲了如下三种：

- 连续性
    
- 可微性（微积分学的支柱之一）
    
- 可积性（微积分学的支柱之二）
    

  

对于每一个性质，都有三个问题需要解答：

1. 是什么？（这个性质是如何定义的）
    
2. 什么是？（怎么样才能满足这个性质）
    
3. 是又怎么样？（有了这个性质，有啥特别的或者有啥用）
    

  

  

4.2.1 连续性

是什么：通过极限定义何为在一点连续，在整个定义域连续则为连续函数。

什么是：通过定义就可以帮助判断函数在一点是否连续，从而是否为连续函数。

是又怎么样：

  

- 连续函数的四种运算（映射的四种运算）都能保持连续性。
    
- 连续函数保持收敛性，也即极限与函数可交换次序。
    
- 连续函数保持紧致性，定义域紧致那么值域也紧致。依此有最大值原理—紧致集合上，连续函数在某点取最大值。
    
- 连续函数保持连通性，中值定理—连通定义域上，若某值在值域中两值之间，定义域存在一点可得此值。
    
- 在紧致集合上，连续函数还具有一致连续性。
    

4.2.2 可微性

是什么：通过极限定义何为在一点可微，在整个定义域可微则为可微函数。【注：在分析中，最好杜绝采用几何概念（如斜率来定义导数）这种交叉思考的行为，几何直观虽然有一定意义，但在分析中，直观并不总能起作用。】

  

什么是：通过定义就可以帮助判断函数在一点是否可微，从而是否为可微函数。

  

是又怎么样：

1. 通用微分法则，在同一点都可微的函数加减乘除运算 、数乘、复合（链法则）。反函数导数法则。
    
2. 导数的用途（微分中值定理系列，由导数性质来研究函数性质）：
    

  

- 判定极值和函数方向，一阶导数为零，二阶导数正负，用多个中值定理可证明；
    
- 单调性，函数在一点可微，且导数严格正，则函数严格单调，拉格朗日定理；
    
- 洛必达法则，求型或者型极限，柯西定理；
    
- 泰勒定理在近似计算中的应用。
    

4.2.3 可积性

不定积分

先谈一下不定积分，它的定义比较简单。直接从微分的逆运算入手，一个函数的微分运算可获得导函数，而不定积分则为“导函数”的原函数族。

定积分

是什么：平时说的定积分就是Riemann积分，定积分是一个集合到一个数的映射，如果不想涉及面积之类几何方面概念的话，定积分的定义还是得铺垫很多东西的。函数Riemann积分定义为如果函数上下Riemann积分相同，称为Riemann可积，上下Riemann积分是通过逐段常值函数逼近的。

什么是：通过Riemann积分的定义来判断是否可积是复杂和困难的，不过对于一大类函数却是有一些重要结论的：

- 有界区间上，一致连续函数，是可积的。
    
- 有界区间上，（逐段）连续且有界函数，是可积的；闭区间上，连续函数，是可积的。
    
- 有界区间上，单调且有界函数，是可积的；闭区间上，单调函数，是可积的。
    

是又怎么样：

1. 通用积分法则（在同一定义域都可积，部分给出积分计算公式，部分只给出可积性）：

  

- 可积函数的加减和数乘，有积分计算公式。
    
- 可积函数的乘法、max和min、取绝对值，是可积的。
    
- 可积函数非负，定积分非负。
    
- 定义域可拆分，分别积分。
    

  

2. 定积分的用途：

- 积分中值定理，积分与函数值的关系，可以帮助去掉一层积分号，在一些证明中有用。
    
- 几何问题，求曲线所围面积、求光滑曲线弧长（求长公式）、曲率（曲率公式）。
    
- 工程、力学中常用的微元法，定积分本身就是从经典力学而来。
    
- 定积分的近似计算，直接运用定积分定义。
    

可积条件要求定义域和值域都是有界的，从定积分的无限求和的几何观点来看，这是必然的。而有时还需要考虑到定义域无界或值域无界的积分，这两种情形都称为反常积分，其是用极限和Riemann积分的组合定义的，或者说其本身就是可变限Riemann积分的极限运算而已。

4.2.3 微积分基本定理（微分和积分的联系）

1. 微积分第一基本定理：不定积分与微分的联系，连续的Riemann可积函数必有原函数（原函数加一个常数构成不定积分）。
    
2. 微积分第二基本定理（牛顿-莱布尼兹公式）：定积分与不定积分的联系，Riemann可积函数的定积分值为其原函数在端点处的差，这个定理提供了计算定积分的一种简单方法。
    

微分和积分联系起来后，微分的法则可以导出有用的积分运算法则，适合定积分和不定积分：

- 分部积分法，由可微函数乘法运算微分法则导出。
    
- 换元积分法，由可微函数复合运算微分法则导出。
    

  

## 

4.3 多元函数（）

欧几里得空间的维度等于1时，退化为一元函数。在一元函数的基础上，许多概念和结论可以直接迁移到多元函数，然而某些概念则需要重新审视。

极限

多元函数有常规的极限定义，称为重极限，与一元函数相同。但当依次考虑某个变元的极限（其他变元看成固定值）时，可以定义累次极限。累次极限与重极限是有联系的，定理：若某个累次极限（有多个）与重极限都存在，必然相等。

  

连续性

所有相关概念和结论与一元函数相同 。

可微性

一元函数导数的定义用到了除法，而有序多元组（用几何的概念就是向量）的除法未定义，不过其乘法是已成功定义的，通过乘法定义的弊端就是无法通过定义来计算导数，只能通过定义来验证某值是否是导数，而且此时需要证明全导数是唯一的。

对于全导数只知道全导数的结构（一个有序多元组），但是不知道具体的计算方法。不过根据一元函数导数定义的方式，只要分母可除，就可照葫芦画瓢来定义，这就是为何会有偏导数，方向导数这些概念。

1. 偏导数，定义为多元函数沿着坐标轴方向的导数。
    
2. 方向导数，定义为多元函数在一点沿着某个方向的导数。
    
3. 全导数，试出来的，将偏导数组成有序多元组代入全导数定义式成立，由于全导数唯一，如此便找出了全导数的计算方法。多元函数可微定理：多元函数所有偏导数都存在且连续，那么函数是可微的（充分条件，这条定理可以验证函数可微性，用定义验证复杂）。
    

具体说来，全导数表达式为，而标量场中有梯度为，其与全导数本质相同，只是一个采用矩阵方式表示，一个用几何概念向量表示。

  

若函数也是有序多元组也即向量函数，值域属于，那么此时全导数表达式如何呢？处理方法是只要对向量函数的分量函数分别处理即可，全导数可用导数矩阵来描述。

在此基础上，有如下几个知识点很重要：

  

- 多元函数链法则，，复合函数复合函数的全导数。
    
- n阶连续可微（函数属于）：所有阶偏导数（包括混合偏导）连续，一元函数有相同概念。Clairaut 定理：二阶连续可微，二阶混合偏导可换序。
    
- 反函数定理，对于，在某点如果方阵可逆，则在此点局部有反函数，且。
    
- 隐函数定理，函数连续可微，在某点满足且，那么存在一个函数满足，在处可微且，用链式法可直接证明。
    

  

可积性

既然多元函数可以对一个变元定义偏导，可以定义方向导数（思路都是当做一元函数处理），那么其对于单变元定义积分也就不奇怪了。

- 含参积分：只留一个变元，其他变元都认为是已知量，那么此时与一元函数的积分完全一致，当然积分出来的结果含有其他变元。
    
- 累次积分：对于不同变元依次进行含参积分，如果每次都可积的话，累次积分存在。
    
- 重积分：Riemann积分的定义也适合多元函数，重积分的概念源自于这样一条定理：多元函数在某定义域上是Riemann可积的，如果累次积分存在，那么累次积分与Riemann积分相同。闭区间上，如果函数连续，那么任何次序的累次积分都存在且相等。
    

  

两类曲线与曲面积分的几何问题其实是多元积分的直接应用，曲线积分是伪装成二元函数积分的一元函数积分，曲面积分是伪装成三元函数积分的二元函数积分。
