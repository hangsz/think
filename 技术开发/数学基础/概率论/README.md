# 独立性检验

问题：两个随机变量是否相互独立

假设：独立，H0

检验：![](https://cdn.nlark.com/yuque/__latex/39c4ddc63776052ab5f4b6c76fcff874.svg)

实际：

|   |   |   |   |
|---|---|---|---|
||分期|不分期||
|男|90|110|200|
|女|30|70|100|
||120|180||

期望期：根据变量x1的总体分布，重新计算另一个变量x2取不同值下的x1的分布

|   |   |   |   |
|---|---|---|---|
||分期|不分期||
|男|80|120|200|
|女|40|60|100|
||120|180||
